# FoodHub - Meal Ordering Platform

A full-stack meal ordering platform with three roles: Customer, Provider, and Admin.

## Features

- Homepage with 4+ sections
- Customer: Browse meals, cart, checkout, order tracking
- Provider: Dashboard, menu management, order fulfillment
- Admin: User management, categories, order oversight

## Tech Stack

**Frontend:** Next.js, TypeScript, Tailwind CSS
**Backend:** Node.js, Express, PostgreSQL, Prisma

## Installation

```bash
npm install
npm run dev
```
